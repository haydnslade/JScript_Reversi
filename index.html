What is Othello?

Othello is a strategy game based on an 8x8 game grid where players consecutively place discs to convert opposing discs, until all places are filled with the player having more discs winning.

Why Reversi/Othello?

Reversi was supposedly created by one of two English men, while each claimed the other to be a fraud. The game did not see major success until the a version of the game was released in the 1970s by the Japanese game company Tsukuda Original registered as "Othello".

You can learn more about Reversi/Othello here.

How to play Othello

Othello is played by two players, each taking turns and placing game pieces, called discs. Each disc must be placed so that it creates a new boundary (horizontal, diagonal, vertical) between another of the same player's discs. Each of the opposing player's owned discs now bounded by two of the current player's discs are converted to the current player. Each player continues placing discs in turns until all discs have been placed, the player that then owns the most discs is declared the winner.

  • The board begins with four total discs placed in the center four positions (two for each player.)
  • The first player (black) now places a disc to create a boundary between the newly placed disc and an existing disc of the same player's. This then converts all opposing discs and they are 'flipped' over to belong to the current player.
  • The next player then repeats the process by placing another disc.
  • Each player continues placing pieces until all board positions have been filled or a player is unable to place a piece that converts any opposing pieces.
  • Once the game has ended the player with the most owned discs is declared the winner.

Preliminary Software Design

As described above, Othello is a two player strategy game played on an 8x8 grid. The main components to the model deal with are the grid, pieces, overall game state, and player information.

Model

Constants

OTHELLO_CONST = { gridSize = 8 the grid size, being gridSize x gridSize.
maxDiscs = gridSize x gridSize the maximum number of playable discs.
numPlayers = 2 number of players.
playerOne = 1 player one enum.
playerTwo = 2 player two enum.
empty empty enum.
playerOneStartPos player one starting positions array.
playerTwoStartPos player two starting positions array.
}

These objects will be used to control the state of the game.

Grid
The Grid will consist of the following attributes:
othelloGrid = {

  • discGrid: array that stores the discs in play, and which player they belong to.
  • initGrid()
    Initializes the grid to a beginning state with player one and two start
  • validPlay(player, x, y)
    Returns if the grid location is a valid play for the specified player.
  • executePlay(player, x, y)
    Attempts to store a new piece at the specified grid location.
  • getPlayerAtPos(x, y)
    Returns the player (or empty) at the specified coords.
  • convertDiscs(player, x, y)
    Converts discs using the location presented and checks all locations that intersect with it to be owned by the player.
  • countDiscs(player)
    Counts the number of discs for the specified player on the board.
}
Game The Game will consist of the following attributes:
othelloState = {
  • discsRemaining: integer of the number of remaining playable discs, before board is full.
  • currentPlayer: the current player, used to determine which piece to place.
  • gameGrid: Grid object that stores the state of the board.
  • initializeGame()
    Initializes the game, such as setting the correct starting pieces and choosing a player to start.
  • nextTurn()
    Changes the active player for the next turn.
}
